Welcome to UW Veterinarian!

This database application was developed by Bob Schmitz, Danielle Lambion and Rick Morrow as a final project for UW Tacoma's TCSS-545, Advanced Databases.

The application represents an initial build of patient and appointment tracking software for use at veterinary clinics. The application's current functionality includes the following:

Listing all patients at a clinic with sorting and filtering
Adding/removing a patient record
Viewing medical records for an individual patient
Viewing appointment records for individual veterinarians
Viewing aggregate data on conditions and allergies

The database for this software his hosted with Amazon RDS, and the API is built using Python's Flask library and Jinja2 templates.

Site icons are obtained from Bootstrap and Font Awesome. Bootstrap is also used for CSS.
